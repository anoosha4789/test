Configuration Reset
This will restore the panel to its factory default configuration.
Make sure you “Export" the current configuration and/or “Download" any existing data and/or log files that you do not want to lose after this reset.
If you reset, the current configuration, as well as all the data and log files will be deleted.
Configuration Reset
This will restore the panel to its factory default configuration and then perform a system shutdown.
Make sure you “Export" the current configuration and/or “Download" any existing data and/or log files that you do not want to lose after this reset.
If you reset, the current configuration, as well as all the data and log files will be deleted.   After reset is completed, the system will shutdown.
